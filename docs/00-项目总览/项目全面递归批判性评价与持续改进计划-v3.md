   - 检查代码示例是否可运行
   - 验证代码示例是否经过测试
   - 评估代码示例质量

4. **第四层：知识缺口分析**（新增）
   - 对标最新技术趋势
   - 识别知识盲区
   - 评估技术前沿性

- **完整性**：内容是否完整，是否有遗漏
- **准确性**：技术内容是否准确，是否有错误

- **一致性**：文档格式是否一致，术语是否统一
- **可读性**：文档是否易读，结构是否清晰
- **实用性**：内容是否实用，示例是否可运行
