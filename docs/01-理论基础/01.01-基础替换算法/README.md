## 核心特性

### 1. LRU算法

- **时间局部性**：最近访问的数据更可能再次访问
- **实现方式**：双向链表 + 哈希表
- **时间复杂度**：O(1)查找、插入、删除

### 2. LFU算法

- **频率局部性**：访问频率高的数据更可能再次访问
- **实现方式**：频率统计 + 优先队列
- **时间复杂度**：O(log n)查找、插入、删除

### 3. FIFO算法

- **队列结构**：先进先出，简单高效
- **实现方式**：队列
- **时间复杂度**：O(1)查找、插入、删除

### 4. MRU算法

- **栈结构**：最近访问的数据优先淘汰
- **实现方式**：栈
- **时间复杂度**：O(1)查找、插入、删除

### 5. Random算法

- **随机替换**：随机选择淘汰项
- **实现方式**：随机数生成
- **时间复杂度**：O(1)查找、插入、删除
