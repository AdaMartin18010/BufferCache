## 核心特性

### 1. ARC算法

- **自适应机制**：根据工作负载自动调整参数
- **双列表机制**：LRU列表和LFU列表
- **竞争比**：O(1)竞争比，接近最优

### 2. LRU-K算法

- **K次访问历史**：记录K次访问历史
- **频率统计**：基于访问频率的淘汰策略
- **性能优化**：K值选择优化

### 3. Clock算法

- **时钟指针**：循环扫描缓存项
- **二次机会**：给访问过的项第二次机会
- **性能优化**：指针移动优化

### 4. W-TinyLFU算法

- **窗口机制**：滑动窗口统计访问频率
- **TinyLFU统计**：紧凑的频率统计
- **性能优化**：窗口大小优化
