## 核心概念

### 缓存层次结构

L1 → L2 → L3 → 主存 → 磁盘，延迟递增，容量递增。

### 缓存一致性

多核系统中保证数据一致性的协议和机制。

### 预取策略

预测性加载数据到缓存，减少缓存缺失。
