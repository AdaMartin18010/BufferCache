# 04-架构设计

## 主题概述

本主题从架构设计角度分析缓存系统的设计模式、行业应用场景和最佳实践，提供架构决策的指导原则。

## 核心概念

### CAP权衡

一致性、可用性、分区容错性的权衡选择。

### 缓存分层

L1本地缓存 → L2分布式缓存 → L3数据库。

### 降级策略

缓存失效时的降级和熔断机制。
